# February: Master Core Languages

## What's This?
A personal February challenge focused on deepening my understanding of core programming concepts by working with multiple languages. The goal is depth over breadth in *fundamentals*—like OOP, algorithms, and clean code—by implementing them in different paradigms.

## Repository Contents
This repo is organized by language/topic:
*   **`/csharp-aoop`**: My deep dive into Advanced Object-Oriented Programming principles using C#.
*   **`/cpp-algorithms`**: Implementing classic Data Structures & Algorithms from scratch in C++ to understand memory and performance.
*   **`/html-css`**: Building small components to practice layout and responsive design fundamentals.
*   **`/python-scripts`** (Optional): Quick Python scripts to experiment with specific features like generators or decorators.

## Guiding Principles
Throughout all this code, I'm actively applying concepts from *Clean Code* by Robert C. Martin:
*   **Readability:** Writing code that is easy to read and understand.
*   **Simplicity:** Keeping functions small and focused on one task.
*   **Maintainability:** Structuring code so it can be easily changed and improved.

## End Goal
By the end of February, I aim to feel confident reading, debugging, and optimizing codebases in any of these languages, with a strong focus on writing professional, maintainable code.
